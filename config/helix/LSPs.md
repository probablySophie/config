# LSPs

## Autocompletion

[Simple Completion Langauge Server](https://github.com/estin/simple-completion-language-server) (`scls`) is a Rust package, so can be installed with Cargo.

```bash
cargo install --locked --git https://github.com/estin/simple-completion-language-server.git
```

* [Configuration in Helix's `languages.toml`](https://github.com/estin/simple-completion-language-server#configure)
* [Creating/Editing Snippets](https://github.com/estin/simple-completion-language-server#snippets)


## Bash

[`bash-language-server`](https://github.com/bash-lsp/bash-language-server) is an NPM package, so install it globally with your favourite package manager.

```bash
npm i -g bash-language-server
bun add -g bash-language-server
```
