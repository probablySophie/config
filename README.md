# config
Config files!  Mostly pushed from my private dotfiles, but these ones are for use elsewhere

The remote/ folder is for devices/servers/machines that I remote into

## Structure


```bash
.
├── config/ 			# Files going in ~/.config
├── install/			# Install scripts
├── local_share/ 		# Files going in ~/.local/share
├── other/  			# Config for random things requiring manual application
├── remote/ 			# Config for remote servers
├── setup/  			# Guides for setting things up the way I like them
├── timeout/			# Files/folders that I'm not happy with
└── create_symlinks.sh 	# Applying the config
```
