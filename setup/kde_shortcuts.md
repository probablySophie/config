# KDE Window Manager Shortcuts

They're edited in `~/.config/kglobalshortcutsrc`

```
[services][Alacritty.desktop]
New=Ctrl+Alt+T

[services][net.local.rofi.desktop]
_launch=Meta+Space

[services][net.local.swaync-client.desktop]
_launch=Meta+Shift+N

[services][org.kde.konsole.desktop]
_launch=none

[services][org.kde.spectacle.desktop]
RectangularRegionScreenShot=Meta+Shift+S
_launch=Print
```
